---
title: HTTP API (XRPC)
---

For client-server and server-server communication we use a set of common conventions called **XRPC** adopted from [AT Protocol](https://atproto.com/specs/xrpc).

All XRPC endpoints and their schema are defined within [Lexicons](/specs/lexicon).

## Libraries

* [@atproto/xrpc](https://github.com/bluesky-social/atproto/tree/main/packages/xrpc) (TypeScript)
